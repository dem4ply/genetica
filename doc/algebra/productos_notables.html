productos notables

binomio conjugado

( a + b ) ( a - b ) = aa - bb

binomio al cuadrado

( a + b ) ** 2 = aa + 2ab + bb ( a - b ) ** 2 = aa - 2ab + bb

binomio al cubo

( a + b ) ** 3 = aaa + 3aab + 3abb + bbb ( a - b ) ** 3 = aaa - 3aab + 3abb - bbb

binomios con termino comun

( x + a ) ( x + b ) = xx + ( a + b )x + ab

binomio de newton

( a + b ) ** n
